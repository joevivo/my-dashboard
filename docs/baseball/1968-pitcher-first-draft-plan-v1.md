# 1968 Strat365 Pitcher-First Draft Plan v1

## Status

Working draft plan for an $80M Strat-O-Matic 365 1968 team.

This is not a UI artifact. It is an analytical draft plan based on:

- 537-player browser-baseline universe
- 156 card-backed players after SS capture/scoring
- Pitcher-first / run-suppression roster posture
- Defensive prioritization at SS, 2B, 3B, CF, RF, and C

## Strategic Posture

Build type: **Pitcher-first / run suppression**

Primary park: **Astrodome 1968**

Fallback park: **Dodger Stadium 1968**

Avoid for this build:

- Oakland Coliseum 1968
- Busch Stadium 1968

Draft thesis:

- Suppress home runs.
- Buy run prevention.
- Use a strong defensive spine.
- Exploit cheap bullpen depth.
- Avoid overpaying for slug-only bats.
- Accept modest offense if the roster commits fully to low-run prevention.

## Known Risks

- Offense may be light.
- Fewer elite bats means the park choice must remain pitcher-friendly.
- SS/3B defense should not be sacrificed casually.
- Some non-card-backed players remain in the broader universe and may need targeted card capture if they become draft-relevant.
- The hybrid model currently uses candidate card weights, not full opponent-card simulation.
- Defensive X-chart outcomes are identified but not yet fully converted into run values.

## Current Recommendation

Draft this team with **Astrodome 1968** as the primary park.

Astrodome gives the roster elite HR suppression while preserving enough singles support for a right-heavy lineup. Dodger Stadium remains the fallback if Astrodome is unavailable, but Dodger's 9/3 singles split is less friendly to this roster's right-handed core.

Do not drift toward Wrigley/Crosley-style offense after selecting this roster. The roster works only if the park, pitching, and defense all point in the same direction.
