# Investigation Checklist

## Required Sections

- [ ] Question
- [ ] Entity
- [ ] Identity
- [ ] Evidence
- [ ] Facts
- [ ] Hypotheses
- [ ] Insights
- [ ] Confidence
- [ ] Reasoning Trace
- [ ] Open Questions
- [ ] Suggested Investigations

## Rules

- Evidence must include provenance.
- Facts must be directly supported by evidence.
- Hypotheses must reference supporting facts.
- Insights must reference supporting hypotheses.
- Confidence must explain uncertainty.
- Reasoning Trace must explain major inference steps.
- Open Questions must identify meaningful uncertainty.
- Suggested Investigations must explain why they are recommended.

## Definition of Done

An endpoint should not claim to return an Investigation unless it satisfies this checklist.
